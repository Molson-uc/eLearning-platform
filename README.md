# E-learning platform
## Introduction
Goal of this project is to improve mu django skills.
It is platform where you can find courses about Tabletop games.

## Main features
- Python 3.8
- Django
- Docker
- Postgres

## Setup
```
git clone https://github.com/Molson-uc/eLearning-platform.git
docker compose up
go to http://0.0.0.0:8000/
```
